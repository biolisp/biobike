User Sessions and Logs


As you continue to work with the VPL, you may come to a point where you wish to save your work, say after defining several important functions and variables.

You can do this by selecting File --> Save user sesssion

The box you see on top is a list of your currently saved sessions, so you don't accidentally overwrite one.

The box beneath is where you will enter the session identifier.

Once you have saved your session, you can exit the VPL or log out of the weblistener, come back later, and select
File --> Restore user session.
A list of currently available sessions will pop up, and you can pick which one to restore.

There is no limit on the amount of sessions you can save.


It's also very easy to see logs of everything you've executed in the weblistener or VPL. Simply bring down
File --> Logs
Or, for a simpler method of seeing just what you've done in the VPL since you last logged in, bring down
File -- > Show execution log
