GREEN BUTTONS

FUNCTION CATEGORIES

Functions are categorized so that you might find what you're looking for by browsing. In the example shown, functions related to genes and proteins are gathered together under a single menu. If you were looking for a means of displaying the annotation of a gene, you might guess (correctly) that DESCRIPTION-OF would be appropriate.

Mousing over any item with a sideways triangle opens up a submenu that offers more choices. Clicking any item that is not a submenu causes the function to be brought down into the workspace.

ALPHABETICAL LIST

If you know the name of the function you're looking for, it may be faster to pick it out from the alphabetical list of all functions, using the ALL menu.

Functions whose names do not begin with letters (e.g. !, factorial) may be found in the first category, symbols.

FUNCTION DOCUMENTATION

Most items on all the menus are followed by ?. Mousing over the question mark pops up a brief description of the function.

Clicking either the question mark or the brief description brings you to a more complete description of how to use the function, usually including examples.

FINDING FUNCTIONS

If you don't know the name of the function, you still may be able to find what you are looking for by describing it in the Help box in the upper right corner of the workspace. Type descriptive words, then press Enter.
