Sometimes when using the VPL you will encounter an error pop up window.
That's okay, we'll talk about them here.

Simply fill in the the empty holes -- marked by a dashed border -- and then execute the code.

Some functions will accept arguments only of certain types. Trying to pass a string to the + function...

...results in an error:
Ruh Roh. User error... Cannot insert "A" here! Only values of type NUMBER are allowed.

which results in the error:
