BLACK BUTTONS

HELP menu
Tools to help you figure out how to make use of BioBIKE
  • How to get started: Offers possibly useful first things to do
  • Ways to get help: Points you to resources to think about when things go wrong
  • Interface Tutorial: This gets you to the pages you're reading.
  • Examples: Lists several instances of how BioBIKE addresses real problems
  • BioBIKE - The Manual: Provides a detailed view of the resource works.
  • Known bugs/workarounds: Describes problems others have found and how to get around them.
  • Report problem/Ask for help: Provides a means for you to contact knowledgeable humans when something has gone wrong or you don't know how to do something.
  • What's new?: A blog that describes changes in BioBIKE.

FILES menu
Tools to help you access files and records of your work
  • Files - yours: Presents a directory of your files on BioBIKE
  • Files - shared: Presents a directory of communal files
  • Files - upload: Allows you to bring a file into your BioBIKE directory
  • User contributed stuff: Offers possibly useful variables and functions users have contributed
  • Exit: Leave without closing the session
  • Exit and logout: Closes the session. It will not be available the next time you login.
  • Exit stage left: Just what it says

SESSIONS menu
Tools related to your current session and past and future sessions
  • Execution log - current: Gives you a summary of what you've done this session
  • Execution logs - all: Brings you to summaries of all of your sessions
  • Workspace - save: Saves your work so you can retrieve it later (see workspaces)
  • Workspace - share: Saves your work so others can retrieve it later
  • Workspaces - list: Shows all available workspaces (see workspaces)
  • New session: Opens a new BioBIKE session (without closing the current session)
  • New session (weblistener): Opens a text-based session
  • Join another session: Collaborate with a colleague
  • Preferences: Sets values that control different aspects of your session

EDIT menu
Tools related to modifying the workspace and results areas
  • Paste to toplevel: Pastes whatever is in the BioBIKE clipboard to the workspace (see Copying)
  • New data box: Opens an empty box. This may be useful for monitoring the value of a variable as it changes.
  • Workspace - clear: Removes all boxes from the workspace. Good for getting rid of the clutter.
  • Workspace - reverse order: Reverses the order of boxes on the workspace. If you prefer most recent and all subsequent boxes to appear at the top of the workspace, try this out.
  • Results - clear: Removes all boxes (except current) from the results. Clicking it again, removes all the results.
  • Results - reverse order: By default, the most recent result is on top. Click this to reverse this policy.
  • Redraw all: If the display somehow gets mangled, try this out.
