Function READ-FASTA-FILE example & mini-tutorial "how-to-upload files to BioBIKE"

Suppose you want to export this example file to BioBIKE.

  1. Save this FASTA file as "example-seq.txt"

  2. Go to WebListener --> Tools --> Upload File

  3. Browse/select the file you want to upload




  4. Enter the directory where you want to put the file in. If you don't specify the directory, the file will go into the default directory. Examples:
    1. Type "example-seq.txt" in the box -> will upload the file in the default directory
    2. Type "/home/visitors/shared-files/example-seq.txt" -> will upload the file in the specified directory

  5. Hit [Upload it] button, and you'll see this screen




  6. Go back to WebListener, and type in the box
    (READ-FASTA-FILE "example-seq.txt")

  7. If the file is in the shared directory, type:
    (READ-FASTA-FILE "example-seq.txt" shared)
