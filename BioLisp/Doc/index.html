News!

  • 20090511: A new BioBike paper, entitled "BioBIKE: A Web-based, Programmable, Integrated Biological Knowledge Base" (by Jeff Elhai, Arnaud Taton, and others) appears in a special issue of Nucleic Acids Research (NAR) on biocomputational web servers. This paper focuses on the VPL and other accomplishments since the last major paper (the 2005 BioInformatics paper). Click here to access this Open Source paper.
  • 20080302: A new BioBike Google Group has been created. Anyone can join it. Click [here] to visit the new group.
  • 20071205: The paper "The Evolution of BioBike: Community Adaptation of a Biocomputing Platform " has just appeared in Studies in History and Philosophy of Science (v38, pp642-656). Ask Jeff Shrager for a PDF of the final version that appeared in the journal.
  • 20071025: A new tutorial on Lisp Macros has been added to the Live Tutorials set. You can see it by [clicking here], but it's really supposed to be run live from inside BioBike. (See instructions below on how to do this.)
  • 20070404: The latest BioBike paper just appeared in the Open Source journal PLoS One; Deductive BioComputing (Shrager, Waldinger, Stickel, and Massar) describes the way that SRI's SNARK theorem prover can be used inside the BioBike platform to do complex biological reasoning.
  • 20061215: A walk through tour and engineering-level description of the new BBL visual programming language (vBBL) has been posted. This is more-or-less a teaser as vBBL hasn't been released yet, but it gives you some idea of how it looks and works.
  • 20060904: A new "success story" about BioBike just appeared on the LispWorks web site
  • 20060830: I've just completed yet another BioBike paper, this one for a special issue of the Journal of the History and Philosophy of Science. Here's a preprint in PDF.
  • 20060815: Richard Waldinger, Mark Stickel, JP Massar and I just completed a new paper on "Deductive Biocomputing". This describes using SNARK, SRI's First Order theorem prover, combined with BioBike to enable "intelligent" (deduction-based) biological computation. Here's a preprint in PDF.
  • 20060606: Richard Waldinger (SRI) presented a paper entitled "Deductive Discovery and Composition of Resources" which describes BioDeducta (BioBike + Snark). The paper appears in the proceedings of the conference: Reasoning on the Web 2006 (RoW06). Here's a link to the conference proceedings pdf for that paper.
  • 20060228: Numerous biobike tutorials are now available! Many of these are Live Tutorials, which are intended to be run live from within BioBike.
  • 20050523: A paper describing the greater "KnowOS" (a knowledge operating system) vision (of which BioBike is the main instance) was delivered at the International Lisp Conference at Stanford in June. See About BioBike above for a link to the paper.
  • 20050606: A commentary appearing in the June 6, 2005 issue of The Scientist describes our collaboration with the HyBrow team at Penn State. (This content is no longer online at The Scientist.)
  • 20050523: A paper describing the greater "KnowOS" vision (of which BioBike is the main instance) will be delivered at the International Lisp Conference at Stanford in June. See About BioBike below for a link to the paper.

Live Tutorials:

These are intended to be run live from within BioBike. To get to the Live versions of Live Tutorials, log into BioBike demo server and select "Help...", then "Live Tutorials", and choose from the list list of live tutorials that appears. Outside a running BioBike, these are just normal HTML pages, but they may not fully work if you just read them straight from here; for example some images might not appear correctly. Also, please note that some of these are still under construction.
  • A Tour of BioBike: Getting one's feet wet with BioLisp and the BioBike Listener
  • Lisp I: Evaluation: The basics of Lisp evaluation.
  • Lisp II: Data and Iteration: Lisp data structures and iteration (loops).
  • How Blast Works: How sequence alignment (i.e., Blast) works.
  • Functions: This is what makes Lisp Special!
  • Lisp Macros: How to Define Entirely New Languages in Lisp
  • Tables: Working with data tables
  • Biocomputing I: Comapring gene function across organisms using annotation and microarrays
  • Graph R&R: Representation and Reasoning about Graphs
  • GO Analysis I: Simple symbolic biocomputing based on the Gene Ontology
  • Microarray Analysis: Simple microarray data clustering
  • Metabolic Simulation I: Introduction to dynamic analysis of metabolic pathways
  • Metabolic Simulation II: Dynamic pathway analysis using the Gene Ontology knowledge base
  • Regulation Simulation: Simulating the cell cycle (based upon a 2004 PNAS publication)
  • Natural Language I: Extracting biological relationships from PubMed I
  • Functional Phylogeny: Exploring phylogeny based upon gene function
  • Model Discovery I: Reverse engineering of regulatory networks
  • Reasoning Intro: Introduction to automated reasoning using Snark
  • Basic BioReasoning: Re-understanding the Elhai problem as a reasoning
  • Regulatory Reasoning: Using Snark to reason about the cell cycle (not done!)
