BioBIKE
(IN-VITRO-TRANSLATE RNA-sequence )
  • Takes an RNA sequence and translates it, according to alien rules
  • Translation may be initiated anywhere within the sequence (no need for a start codon)
  • Returns a list of peptides resulting from 100 translations
  • The peptides are represented using 1-letter codes for amino acids
  • RNA may be conveniently generated by using MAKE-RANDOM-RNA
Examples:
  (MAKE-RANDOM-RNA "UC")
  :: "UCUCUCUCUCUCUCUCUCUCUC..."
  (IN-VITRO-TRANSLATE *)
  :: ("QRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQR..."
      "QRQRQRQRQRQRQRQRQRQR"
      "RQRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQ..."
      ...)
