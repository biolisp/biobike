BioBIKE
(MAKE-RANDOM-RNA letter )
(MAKE-RANDOM-RNA letters )
(MAKE-RANDOM-RNA list-of-letters )
  • Returns an RNA sequence constructed at random but directed by the input letter[s] or list of letters
  • If a letter is given (between quotation marks), then the function returns 1000 nucleotides of that letter
  • If multiple letters are given (between quotation markes), then the function returns at least 1000 nucleotides of that letter combination
  • If a list of letters is given, then the function returns 1000 nucleotides, randomly selecting from the list
Examples:
  (MAKE-RANDOM-RNA "U")
  :: "UUUUUUUUUUUUUUUUUUUUUU..."

  (MAKE-RANDOM-RNA "UC")
  :: "UCUCUCUCUCUCUCUCUCUCUC..."

  (MAKE-RANDOM-RNA {"U" "C" "C" "C"} )        ; in BioBIKE mode
  (MAKE-RANDOM-RNA (LIST "U" "C" "C" "C") )   ; in BioLISP mode
  :: "CCCAACCCCACCCCCACCCCCC..."
