BioBIKE
(ANALYZE-AMINO-ACID-CONTENT list-of-peptides )
  • Counts amino acids found within a list of peptides
  • The amino acids are displayed using 3-letter codes for amino acids
  • The list of peptides may be conveniently generated by using IN-VITRO-TRANSLATE
Examples:
  (MAKE-RANDOM-RNA "UC")
  :: "UCUCUCUCUCUCUCUCUCUCUC..."
  (IN-VITRO-TRANSLATE *)
  :: ("QRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQR..."
      "QRQRQRQRQRQRQRQRQRQR"
      "RQRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQ..."
      ...)
  (ANALYZE-AMINO-ACID-CONTENT *)
  ::
  GLN 8562
  ARG 8554
