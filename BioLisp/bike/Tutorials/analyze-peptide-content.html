BioBIKE
(ANALYZE-PEPTIDE-ACID-CONTENTlist-of-peptides )
  • Counts peptides found within a list of peptides
  • The peptides are displayed using 1-letter codes for amino acids
  • Only the first 12 amino acids of each peptide are considered
  • The list of peptides may be conveniently generated by using IN-VITRO-TRANSLATE
Examples:
  (MAKE-RANDOM-RNA "UC")
  :: "UCUCUCUCUCUCUCUCUCUCUC..."
  (IN-VITRO-TRANSLATE *)
  :: ("QRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQR..."
      "QRQRQRQRQRQRQRQRQRQR"
      "RQRQRQRQRQRQRQRQRQRQRQRQRQRQRQRQ..."
      ...)
  (ANALYZE-PEPTIDE-CONTENT *)
  ::
  QRQRQRQRQRQR... 44
  RQRQRQR 1
  RQRQRQRQRQRQ... 56
